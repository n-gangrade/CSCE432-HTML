My accessibility interests

I’m interested in accessible computing because I want products to work well for more people, in more contexts. I’m especially curious about:

  1. Designing pages that work with screen readers and keyboard navigation
  2. Clear structure: headings, landmarks, and meaningful link text
  3. Accessible visual design (contrast, spacing, readable typography)

Topics I’d love to learn more about

  • WCAG best practices applied to everyday web apps
  • Accessible data visualizations and alt-text that’s actually useful

How we might work well together

If you’re working on a team project and you care about accessibility, I’d love to collaborate on:

  • Creating accessible page structure (semantic HTML + ARIA when needed)
  • Writing meaningful alt text and testing keyboard navigation
  • Designing forms that are easy to understand and complete

If you’re into inclusive design, usability testing, or building accessible web components, let’s talk.
